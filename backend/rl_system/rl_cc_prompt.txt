Build me a **super simple, modular local PPO reinforcement learning prototype** in Python for my research project.

## Project goal

I am building a system where:

1. a language model generates a response to a user prompt
2. a separate reward system collects **user-defined trait scores**
3. those trait scores are converted into **one scalar reward**
4. PPO uses that scalar reward to improve the model

This module is only for the **reinforcement learning / PPO side**.

The two most important parts of the full project are:

- **human feedback / user scoring**
- **PPO training**

Do **not** overengineer this.
Do **not** build a polished frontend.
Do **not** build the reward system here.
Assume the reward system already gives me:
- prompt
- generated response
- scalar reward

I want the **simplest clean PPO prototype** that can be joined later to the rest of the project.

---

## Very important project context

The reward traits are **user-defined**, not hardcoded.

That means PPO should **not** assume fixed reward categories like:
- toxicity
- helpfulness
- warmth

PPO only receives the **final scalar reward** computed elsewhere.

So this module should treat the reward as an external number.

Build the **smallest clean local implementation** of a PPO training module that can later connect to:

## Core idea

The PPO module should accept data in a form like:

[

    "reward": 4.1

This data is assumed to come from the generation + reward pipeline later.

For now, create a tiny local demo dataset manually.
