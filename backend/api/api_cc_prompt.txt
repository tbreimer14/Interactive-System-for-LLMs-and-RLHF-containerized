Build me a **super simple backend API** for my local research project.

==================================================
PROJECT CONTEXT
==================================================

I am building an interactive LLM system with:

1. RAG system
- retrieves chunks from local data
- builds prompt
- generates response
- retrieval and generation are separate modules

2. Reward / feedback system
- user defines traits (NOT hardcoded)
- examples: clarity, empathy, humor, etc.
- each trait has:
    - weight
    - score
- scalar reward:
    reward = sum(weight_i * score_i)

3. PPO later (NOT NOW)
- only log data for future training
- do not implement training

==================================================
API ENDPOINTS
==================================================

1. Health check
GET /health

Response:
{
  "status": "ok"
}

--------------------------------------------------

2. Generate
POST /generate

Input:
{
  "prompt": "text",
  "top_k": 3,
  "show_prompt": false
}

Behavior:
- call backend adapter
- retrieve chunks
- generate response

Response:
{
  "prompt": "...",
  "retrieved_chunks": [
    {
      "id": "chunk_1",
      "source": "file.txt",
      "text": "..."
    }
  ],
  "response": "...",
  "final_prompt": "..."
}

--------------------------------------------------

3. Compute reward
POST /reward/compute

Input:
{
  "traits": [
    { "name": "clarity", "weight": 1.0, "score": 4.0 }
  ]
}

Behavior:
- contribution = weight * score
- sum all contributions

Response:
{
  "traits": [
    {
      "name": "clarity",
      "weight": 1.0,
      "score": 4.0,
      "contribution": 4.0
    }
  ],
  "scalar_reward": 4.0
}

--------------------------------------------------

4. Save interaction
POST /interactions

Input includes:
- prompt
- retrieved_chunks
- response
- final_prompt
- traits
- scalar_reward

Behavior:
- generate ID
- generate timestamp
- append to JSONL file

Response:
{
  "status": "saved"
}

--------------------------------------------------

5. Get interactions
GET /interactions?limit=20

Returns recent interactions

--------------------------------------------------

6. Get single interaction
GET /interactions/{id}
